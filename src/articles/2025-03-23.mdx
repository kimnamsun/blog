포켓몬 게임을 하다가 포켓몬 관련 토이 프로젝트를 하나 해보고 싶어졌다.
너무 복잡한 건 부담스러워서 주말 하루 정도 투자해서 가볍게 만들 수 있는 걸 찾다가, 예전에 봤던 PokeAPI가 떠올랐다.
처음엔 흔한 포켓몬 도감을 만들어볼까 했는데, 이미 비슷한 프로젝트가 많았다.
그래서 다른 방향을 고민하다가, 포켓몬 카드를 3D로 만들어보면 재밌겠다는 생각이 들었다.
평면적인 이미지 대신 입체적으로 만들면 더 흥미로울 것 같았다.

그러면서 Three.js를 알게 됐고, React와 함께 쓰기 좋은 `@react-three/fiber`와 `@react-three/drei`도 찾아보게 됐다.
적당히 써보기에 괜찮아 보여서 이걸로 만들어보기로 했다.

## Three.js가 뭔데?

Three.js는 웹에서 3D 그래픽을 쉽게 만들 수 있게 해주는 자바스크립트 라이브러리다.
처음엔 엄청 어려울 줄 알았는데, 찾아보니까 WebGL 기반으로 동작하면서도 훨씬 직관적으로 사용할 수 있는 도구였다.
하지만 직접 쓰는 대신 `@react-three/fiber`를 선택했다. React 컴포넌트처럼 Three.js를 다룰 수 있는 라이브러리라서 React를 자주 쓰는 내게 더 익숙했다.
여기에 `@react-three/drei`도 같이 활용했는데, `<OrbitControls>`나 `<Html>` 같은 자주 쓰이는 유틸리티를 바로 제공해줘서 작업 속도가 훨씬 빨라졌다.
카메라, 조명, 오브젝트를 설정해 3D 공간을 만들고, 마우스로 회전하거나 애니메이션을 추가하는 것도 가능하다.

## 프로젝트 구상

아이디어는 간단했다. 포켓몬 카드 모양을 3D로 만들고, 그 안에 포켓몬 정보(이름, 타입, 이미지 등)를 넣는 것.
주말 하루 목표라서 너무 거창하게 잡지 않고, 이 정도만 해도 재밌을 것 같았다.

### 프로젝트 기본 구조

Three.js만 사용하면 HTML 요소를 조작하는 게 불편할 것 같아,
React와 Three.js를 함께 사용하는 방식으로 진행했다.
Three.js의 `<Html>` 컴포넌트를 사용하면 3D 객체에 2D 콘텐츠를 붙이는 것도 가능하다는 걸 알게 됐다.

### 기본 플로우

Three.js로 카드의 기본 형태(살짝 입체감 있는 박스)를 만든다.

React로 포켓몬 정보를 표시할 UI를 구성한다.

카드가 회전하거나 마우스로 조작할 수 있도록 애니메이션을 추가한다.

이렇게 하면 단순한 이미지가 아니라, 인터랙티브한 3D 포켓몬 카드가 완성될 것 같았다.

## 1. 카드 만들기.

카드 모양은 기본적으로 평면 형태의 `mesh`로 만들 수 있다.

```jsx
import { Canvas } from "@react-three/fiber";
import { OrbitControls } from "@react-three/drei";

const Card = () => {
  return (
    <mesh>
      <planeGeometry args={[2, 3]} />
      <meshStandardMaterial color="white" />
    </mesh>
  );
};

export default Card;
```

planeGeometry의 args를 이용해 카드의 크기를 조절할 수 있다.
