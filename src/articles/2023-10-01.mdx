## 재시작 후 텍스트 모드로밖에 진입이 안되는 문제 상황

### 1. 사용가능한 패키지 그룹 확인하기

```bash
dnf group list
```

![](https://velog.velcdn.com/images/nsunny0908/post/eafffe0e-b430-49a5-8c08-d2e2def2ccf7/image.png)

### 2. GUI 패키지 그룹 설치

```bash
dnf groupinstall "Server with GUI"

OR

dnf groupinstall "Workstation"
```

### 3. 부팅 설정을 graphical로 변경

```bash
systemctl set-default graphical.target
```

- 기본 부팅 설정을 확인하고 싶다면 아래 명령어로 확인할 수 있다.

  ```bash
  systemctl get-default
  ```

### 4. 재부팅

```bash
reboot
```

## 한글 설정하기

### 1. 변경가능한 locale 확인

```bash
locale -a
```

![](https://velog.velcdn.com/images/nsunny0908/post/f3a591b8-fe38-43c7-8e3b-2aa9f40a3900/image.png)

### 2. locale 변경

```bash
localectl set-locale LANG=ko_KR.utf8
```

- 변경된 locale 확인

